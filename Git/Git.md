# git practice

## git의 세가지 영역
---
1. 프로젝트의 일부가 아닌 것

    -> `git add`

2. 프로젝트에 추가하고자 하는 것

    -> `git commit`

    -> `-m` : 메시지 로그 추가
3. 프로젝트에 추가된 것

## Git 명령어
---
1. `git add` : 커밋할 파일 추가
2. `git commit` :add한 파일을 커밋 (`-m` 안 달았을 시 `:p!`으로 빠져나오기)
3. `git push` : Github과 같은 저장소에 저장
4. `git pull` : 저장소에 저장된 데이터로 업데이트
5. `git clone` : 저장소에 저장된 데이터를 로컬 저장소에 저장
